### PING It Shop

PING It Shop - это ведущий поставщик высококачественных компьютерных компонентов с 2010 года. Наша миссия — предоставлять передовые технологические решения для профессионалов и энтузиастов. Преимуществами PING являются скорость доставки, качество продукции и круглосуточная поддержка.

## Описание

Помимо закрепления основ языка разметки html и языка стилей css целью данной работы также являлось изучение и использование в разработке следующих составляющих:
- Анимации с помощью псевдо-классов
- Переадрессация на другие страницы 
- Реализация адаптивности под телефоны и разную ширину монитора
- Flexbox и grid для отображения элементов на странице

На сайте присутствует множество кнопок, реализация которым не была добавлена в рамках данного проекта, так как для сдачи работы было достаточно показать умение создания хотя бы двух взаимосвязанных страниц. Перечень работающих кнопок представлен ниже:
- Основная навигация по сайту в меню сверху (за исключением выбора отдельной категории товара)
- Переход на страницу покупки процессора i5

Вся информация, представленная на сайте является выдуманной/недействительной/взятой из открытого доступа.

Проект был создан в качестве домашнего задания для курсов дополнительного образования от IT-академии Samsung по фуллстек разработке.

##Установка

Для запуска проекта достаточно выполнить одно из двух:

- Опробовать интернет-ресурс через ссылку на git-deploy
- Клонировать репозиторий и открыть файл "index.html"
